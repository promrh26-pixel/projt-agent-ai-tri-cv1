# AI Agent for CV Screening

## Project Description
This project is an AI prototype that automatically analyzes and ranks CVs during a recruitment process. The goal is to help recruiters quickly identify the most relevant candidates.

## Features
The AI agent can:
- Analyze CV text
- Extract important information
- Evaluate candidates using a scoring system
- Rank candidates based on their qualifications

## Extracted Features
The system extracts the following information from CVs:
- Number of years of experience
- Education level
- Technical skills
- Languages spoken
- Number of previous companies

## Technologies Used
- Python
- Streamlit
- Pandas
- NLP techniques

## How to Run the Project

1. Install dependencies

pip install -r requirements.txt

2. Run the application

streamlit run app.py

## Ethical Considerations
AI systems used in recruitment can introduce bias. It is important to ensure fairness and transparency in candidate evaluation.
